The 5 Tags I Have Learnt for Creating Tables:

Table Tag
To add a table to the page we start with the table tag.
Caption Tag
Every table can have a caption, this summarizes the data in the table. The caption can be added to the table by using the caption paired tag.
Thead Tag
The header of the table is used to add a heading row for the table columns. This is where we add labels for the data being displayed. Using the thead tag.
Tr Tag
We add rows of data to the table using the tr tag for each row. Starting from the first row added inside the thead.
Th Tag
The table heading cell tag th defines the heading cell. We add the heading cells inside the table rows.
